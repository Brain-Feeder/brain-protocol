# Archive

Historical material, preserved for the record. **Nothing here is current** —
the live suite is [`../v2/`](../v2/INDEX.md) and the repository's public face
is the [root README](../README.md).

## `v0.1/`

The Brain Protocol v0.1 reference package as it existed before the v2 suite:
the original `@brainfeed/protocol` TypeScript package (Zod schemas, validators,
negotiation), its conformance kit, and the connector onboarding material. The
v0.1 spec document itself (`BRAIN_PROTOCOL.md`) lived in the Brainfeeder repo's
docs, not here, so some internal references in these files point outside this
repository.

v0.1 peers remain valid partners via negotiation (BP-03 §3.2); the migration
path from v0.1 to v2 Class D is BP-09 §6. Do not build anything new against
this directory.
